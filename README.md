# 沪深300多因子选股系统（毕业设计项目）

## 项目概述
本项目是基于前后端分离架构的沪深300多因子选股系统，采用"行业中性化+XGBoost"算法，实现量化选股模型的构建与可视化展示。

## 技术架构
- **量化核心层**：Python 3.9 + 量化算法
- **后端API层**：FastAPI + RESTful接口
- **前端可视化层**：Vue3 + Element Plus + ECharts

## 项目结构
```
Aver130/
├── quant_core/          # 量化核心模块
├── backend/             # 后端API模块  
├── frontend/            # 前端可视化模块
├── docs/               # 配套文档
└── data/               # 数据文件
```
